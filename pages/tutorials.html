Базові Поради Для Початківців

Спорядження для Freeride

   Щоб фрірайд був безпечнішим та ефективнішим, потрібне правильне обладнання:

     1. Велосипед

      Тип: фрірайдовий MTB або ендуро.

      Ход підвісу: 160–200 мм.

      Колеса: 26" / 27.5" (іноді 29" для ендуро-фрірайду).

      Гальма: гідравлічні дискові, 4-поршневі.

      Покришки: агресивний протектор, ширина 2.3"–2.6".

     2.Захист

      Шолом: обов’язково фулфейс.

      Налокітники / наколінники.

      Черепаха або жилет для спини.

      Захист гомілок (особливо для райдерів, що роблять трюки).

      Рукавички із посиленим захистом.

Базова Техніка Катання

  • Розтяжка перед початком катки
  • Атакуюча позиція тіла
  • Лікті та коліна зігнуті
  • Центр ваги більше назад
  • Тормозити треба не різко, рівномірно щоб не хитало вперед-назад
  • Завжди дивитись вперед наскільки це можливо, щоб сканувати трасу за 10-20 метрів завчасно
